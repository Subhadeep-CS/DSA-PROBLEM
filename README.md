## 🚀 Getting Started

### Clone the Repository
```bash
git clone <repository-url>
cd DSA-PROBLEM
```

### Run a Solution
```bash
# Run array problems
node array/01-count-negatives.js
node array/02-largest-number.js
node array/03-search-element.js
node array/04-second-largest.js
node array/05-smallest-number.js
```
